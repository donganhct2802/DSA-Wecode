# DSA-Wecode
Hello chị! Em tổng hợp lại các bài tập cô gửi từ Wecode ở đây nha.

##  Cấu trúc mỗi bài tập
Để chị dễ xem, mỗi bài em chia làm 3 phần:
1. **Đề bài:** Giải thích yêu cầu, Input và Output.
2. **Template:** Khung code có sẵn phần nhập/xuất trên Wecode.
3. **Solution:** Code hoàn chỉnh để chị tham khảo sau khi đã tự làm xong (hên xui).
